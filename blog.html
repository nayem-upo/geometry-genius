The meaning of DOM is Document Object Model. It's a programming interface. We use DOM to find a element and change the document style.

I use some methods to select HTML elements using DOM and the name of the methods are-
getElementById(), getElementsByClassName() , getElementsByTagName() , querySelector() , querySelectorAll()
